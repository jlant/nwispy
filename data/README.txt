DESCRIPTION

	data/datafiles/ is a directory that contains some sample files to test or use nwispy.py.  
	
	The data files include some clean and dirty (missing data, text mixed with floats)NWIS data files.
	
	The USGS NWIS data files were retrieved from: 
	
		http://waterdata.usgs.gov/nwis/rt
